# 🌬️ Vaayu AI

Vaayu AI is an intelligent air quality forecasting platform that predicts AQI using deep learning.

## Features

- LSTM-based AQI prediction
- 6 / 12 / 24 hour forecasts
- 7-day outlook simulation
- Pollution heatmaps
- City similarity analytics
- Health risk engine
- Manual AQI predictor
- AI explanation module
- Counterfactual pollution simulator
- Interactive Streamlit dashboard
